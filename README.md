# 👗 Duo Chic — Site de Moda Feminina

> Site institucional com vitrine de produtos desenvolvido para a **Duo Chic**, loja de moda feminina elegante localizada no Shopping Jardim Guadalupe, Rio de Janeiro. Design sofisticado com galeria filtrável, hero slider e integração com WhatsApp.

## 📋 Sobre o projeto

Site desenvolvido para a **Duo Chic**, marca de moda feminina especializada em peças elegantes e sofisticadas. O projeto apresenta a coleção completa da loja com filtros por categoria, hero slider com fotos das peças e formulário de contato, tudo com uma identidade visual refinada e alinhada ao público feminino da marca.

---

## ✨ Funcionalidades

- **Hero slider** com 3 imagens das peças em destaque
- **Galeria de produtos** com filtro por categoria — Todos, Vestidos, Conjuntos e Macacões
- **Cards de produto** com nome, descrição e preço
- **Botão "Ver Detalhes"** em hover sobre cada produto
- **Botão WhatsApp** direto no hero para atendimento imediato
- **Seção Sobre** com história da loja e diferenciais
- **Formulário de contato** com campos de mensagem
- **Rodapé completo** com links rápidos, categorias e informações de contato
- **Layout responsivo** para mobile e desktop
